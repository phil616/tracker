# Task Tracker

根据2024年9月8日的四原则规定，该项目用于跟踪纪录任务和日程。

四原则

1. 持续性
2. 记录性  日程，任务，账单 分别按每月，每季度，每年上报
3. 平滑性
4. 标准性

该项目用于记录任务与日程。
